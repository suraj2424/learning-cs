## Question 1

### Problem Statement
Given an integer array `nums` and an integer `k`, return the total number of continuous subarrays whose sum equals to `k`.

**Difficulty:** Medium

### Example:
**Input:** nums = [1, 1, 1], k = 2
**Output:** 2
**Explaination:** ([1,1] at index 0–1 and 1–2)
